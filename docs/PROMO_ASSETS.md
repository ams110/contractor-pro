## 🔒 القواعد — اقرأها قبل أي توليد

| # | القاعدة |
|---|---------|
| **R1** | **افحص السجلّ أولاً.** قبل توليد أي أصل، ابحث في الجداول أدناه. إن وُجد نفس المفهوم → **لا تُعِد توليده**؛ استعمل الموجود أو عدّله بـimage-to-image. |
| **R2** | **الشاشات حقيقية لا موك‑أب.** أي شاشة تطبيق تُلتقط من المسار الحقيقي: `/demoshot` (بيانات ديمو) للشاشات التي تقرأ من المخزن، أو **حساب تجريبي مبذور** للميزات التي تقرأ من الباكند (بوّابة العامل/الضرائب/المالية). **ممنوع توليد شاشة تطبيق وهمية بالـAI** — لِذا أصول `feat_*` صارت احتياطية فقط. |
| **R3** | **تسمية موحّدة**: `<فئة>_<مفهوم>[_text\|_raw].<ext>`. الفئات: `cine`=سينمائي واقعي · `feat`=بوستر ميزة · `viral`=شخصيات · `real`=شاشة حقيقية · `video`=فيديو. |
| **R4** | **خام + نهائي**: احفظ نسخة **خام** (بلا نص، طبقة قابلة لإعادة الاستخدام) ونسخة **`_text`** (بعنوان/CTA). لإضافة نص: عدّل الخام بـimage-to-image، لا تولّد مشهداً جديداً من الصفر. |
| **R5** | **النص العربي/العبري**: ولّده عبر **Nano Banana Pro** (يكتب عربي سليم) أو طبقة `/adstudio`. تحقّق بصرياً أن الحروف نظيفة 100% — **ممنوع خلط حروف عربية وعبرية** (§19.4 CLAUDE.md). |
| **R6** | **فيديو من صورة موجودة**: حرّك أصلاً من السجلّ (image-to-video، Kling) بدل توليد مشهد جديد — اتساق + توفير. |
| **R7** | **سجّل فوراً**: بعد كل توليد ناجح أضف صفاً للجدول (معرّف + مفهوم + نموذج + برومت مختصر + حالة). |
| **R8** | **استرجاع فيديو Higgsfield** (أداة العرض فيها خلل `aspect_ratio:null`): ابنِ الرابط مباشرة → `https://d8j0ntlcm91z4.cloudfront.net/user_3F5zLFwJDv4CsU85Gdpug2nDs0f/hf_<YYYYMMDD>_<HHMMSS>_<jobid>.mp4` حيث `HHMMSS` = وقت إرسال المهمّة (UTC). |
| **R9** | **التصوير المقطوع الرأس**: لازم `--ignore-certificate-errors` + `ignoreHTTPSErrors:true` (شهادة Supabase)، وأطفِ لافتة الكوكيز (`localStorage.cp_cookie_consent`) وبطاقة الترحيب (`localStorage.cp_onboarded='1'`) وإلا تغطّي شريط التنقّل. |
| **R10** | **نظافة**: لا تترك سكربتات/توكنات/مخرجات مؤقتة بالريبو. أي حساب تجريبي مبذور **يُحذف بالكامل** بعد التصوير (`delete from auth.users` → cascade). |
| **R11** | **النموذج**: صور = `nano_banana_pro` (2K، عربي + جودة) · فيديو = `kling2_6` (9:16، 5ث، `sound:false`). عند اقتراح بريسِت، أعد بـ`declined_preset_id`. |

## 6) 📊 تقرير التكرار/الزوائد (حالة اليوم)

**المجموع**: 19 صورة `ads/raw` + 4 فيديو + 9 شاشات `shots` = **32 أصلاً فعّالاً** (بعد حذف لقطة finance المكسورة).

**التكرار المقصود (سليم — لا تحذفه):**
- أزواج **خام/`_text`** (cine + viral): الخام طبقة لإعادة الاستخدام، الـtext هو الناتج النهائي. مقصود.

**التداخل (نفس الميزة بأكثر من صيغة — وحّد الاستعمال):**
| الميزة | النسخ الموجودة | المعتمد |
|--------|----------------|---------|
| بوّابة العامل | `feat_worker_portal` (موك‑أب) · `real-portal` (حقيقي) · `video_worker_portal` | **`real-portal`** للإعلان · الفيديو للهوك |
| إدارة العمّال | `feat_workers` (موك‑أب) · `real-owner-workers` · `shot-2-workers` (ديمو) | **`real-owner-workers`** |
| الضرائب | `feat_tax` (موك‑أب) · `real-owner-tax` | **`real-owner-tax`** |
| السيولة/اللوحة | `feat_cashforecast` (موك‑أب) · `real-owner-dashboard` · `shot-cashforecast` · `shot-1-dashboard` (ديمو) | **`real-owner-dashboard`** + `shot-cashforecast` |

**زائد فعلاً (مرشّح للأرشفة):** أصول `feat_*` الأربعة (موك‑أب) — صار لها بدائل حقيقية أفضل؛ تُبقى احتياطياً فقط. لقطات الديمو `shot-1-dashboard`/`shot-2-workers` مكرّرة مع نسخها الحقيقية (الحقيقي أفضل).

**حُذف**: `shot-1-finance.png` (محاولة مكسورة فارغة قبل تصليح بذر المالية).

**القاعدة المستخلصة**: لأي ميزة لها شاشة حقيقية (`real-*`)، **لا تولّد موك‑أب AI جديد** لها (R2). ولأي مفهوم بصري بالجدول، **لا تكرّره** (R1).
