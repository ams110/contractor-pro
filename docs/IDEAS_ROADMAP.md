## 🚀 الضربات الكبيرة (Big Swings — ميزات تعرّف فئة)

### 1. الطيّار الآلي المالي (AI CFO / Financial Autopilot)
وكيل ذكي يشتغل بالخلفية على بيانات المصلحة، ويعطي المقاول كل صبح **«٣ قرارات بزرّ واحد»** (وافق / أجّل / تجاهل):
- «حوّلت ₪4,200 افتراضياً لجيب الضريبة — أكّد التحويل؟»
- «٣ عملاء متأخّرين ₪18k — أرسل تذكير واتساب للكل؟»
- «مشروع الكرمل هامشه نزل تحت 8% — ارفع سعرك القادم 12%؟»
يجمع كل المحرّكات (`insights.js`) + طبقة قرار + Claude للصياغة. = مدير مالي حقيقي بجيب المقاول.

### 2. النبوءة — محاكي السيناريوهات (What-if Simulator)
مزالج تفاعلية: «إذا وظّفت عامل كمان؟ إذا تأخّر عميل شهر؟ إذا أخذت مشروع ₪80k؟» → يعيد حساب **السيولة + الضريبة + الربح 6 أشهر قدّام** فوراً بنطاق ثقة. قرار تشغيلي بمستوى شركة كبيرة بيد مقاول فرد.

### 3. مساعد الضرائب الكامل (Tax Co-pilot)
يجمّع السنة، يحضّر تقرير **מע"מ / מס הכנסה / ביטוח לאומי** جاهز للمحاسب بضغطة، يذكّر بالمواعيد (دو"ח תקופתי، דו"ח שנתי)، ويكشف خصومات فايتة. وجع رأس حقيقي لكل עוסק.

### 4. التسعير الذكي (Smart Bid AI)
قبل أخذ مشروع: تُدخل النوع/المساحة/المدة → يقارن بمشاريعك السابقة المشابهة وهوامشها ويقترح سعراً يضمن ربحاً مستهدفاً. «مشاريعك المشابهة ربحت 14% — سعّر هذا بـ₪X لتضمن 18%».

### 5. الرفيق الصوتي (Voice-first capture)
المقاول بالموقع وإيديه وسخة: يحكي «سجّل ٣ عمال اليوم بمشروع الكرمل واشتريت إسمنت بـ800 شيكل» → AI يفهم ويسجّل. إدخال بالصوت بالعربي = ثورة للجمهور الميداني.

## 🧠 ذكاء (AI — Claude)
- المستشار المالي الذكي (شات بالعربي يقرأ الأرقام الحقيقية).
- مولّد رسائل واتساب ذكي (تحصيل/تذكير/عرض حسب السياق).
- التقرير الصباحي الذكي (ملخّص يومي بجملة بشرية).
- تلخيص ذكي للمشروع.

## 💰 فلوس ونموّ
- مولّد عروض أسعار/فواتير PDF فخم + واتساب + رابط دفع.
- أتمتة التحصيل (يبني على `CollectionAging`).
- محفظة مستندات (عقود/تأمينات/رخص) + تنبيه انتهاء صلاحية.
- بوّابة العميل (يشوف تقدّم مشروعه + مدفوعاته).

## 🏗️ ميدان وعمّال
- تسجيل حضور GPS من بوّابة العامل + صور تقدّم يومية.
- تقويم جدولة العمّال على المشاريع + كشف التعارضات.
- مسح فواتير الموردين OCR وتصنيف تلقائي (توسعة `scan-receipt`).

## 📊 لوحة الأدمن (تطوير القائم)
- رادار الهجر (تصنيف حسابات + رسالة استعادة).
- لوحة إيراد متقدّمة (LTV / cohorts / تحويل تجربة→مدفوع).
- أعلام ميزات (Feature Flags) من اللوحة بلا نشر.
- فرص Upsell ذكية («حسابات قاربت حدّ עוסק פטור»).
