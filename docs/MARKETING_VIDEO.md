## 1. الاستراتيجية (ليش هالرسائل بالذات)

من بحث السوق — أقوى محرّكات الشراء للمقاول العربي في إسرائيل مرتّبة:

1. **عصرة السيولة**: الزبون يدفع بعد ٧٠–٨٠ يوم، بس الـ מע"מ والرواتب مستحقّة بين ٥–٢٣ بالشهر. ← الخطّاف.
2. **المرساة السعرية «أرخص من محاسب»**: محاسب ١,٨٠٠₪+/سنة وما بيدير ولا عامل. ← تبرير السعر.
3. **التموضّع الفريد**: أول تطبيق يجمع (عربي ١٠٠٪ + بوّابة عامل يومي + محرّك ضرائب إسرائيلي + موبايل‑أول). ما في منافس بيعمل هيك.
4. **القناة**: تيك‑توك بالعربي (٧١٪ من العرب، والمنافسون غايبون عنه) + واتساب + مجموعات فيسبوك للمقاولين.

**القاعدة الذهبية للريلز**: الخطّاف بأول ٣ ثواني (ألم ملموس)، الحلّ بسرعة، السعر كمرساة، CTA واضح. صامت‑أولاً (كتابة على الشاشة) لأنّ أغلب المشاهدات بدون صوت.

## 7. كيف تُخرِج الفيديو فعلياً (٣ طرق)

### الطريقة أ — تلقائي (الأسرع، بدون أدوات)
```bash
node marketing/record-reel.mjs
# → marketing/out/contractor-pro-reel.webm  (1080×1920، ~28–31s)
```
ارفع الـ WebM مباشرة (إنستغرام/تيك‑توك/فيسبوك تقبلها). لتحويلها MP4 إن توفّر ffmpeg:
```bash
ffmpeg -i marketing/out/contractor-pro-reel.webm -c:v libx264 -pix_fmt yuv420p \
       -r 30 marketing/out/contractor-pro-reel.mp4
```

### الطريقة ب — تسجيل شاشة يدوي (أعلى جودة + للتحكّم بالصوت)
1. افتح `marketing/promo-reel.html` بالمتصفح (Chrome) → اضغط **«إخفاء الواجهة»** ثم **«ملء الشاشة»**.
2. سجّل الشاشة بـ OBS/QuickTime بمقاس **1080×1920**.
3. اضغط **«↻ إعادة»** لبدء لقطة نظيفة، وأوقف التسجيل بعد ~٢٨ ثانية.

### الطريقة ج — تسليم لمحرّر فيديو
سلّمه هالملف (`docs/MARKETING_VIDEO.md`) + لقطات `marketing/shots/` + الفيديو المرجعي. الستوري‑بورد في §٢ يكفيه يبني نسخة احترافية بصوت راوي حقيقي.

## 8. ملاحظات إنتاج

- **المقاسات حسب المنصّة**: ريلز/ستوري/تيك‑توك = **9:16** (1080×1920). فيد فيسبوك = 4:5 أو 1:1. اليوتيوب شورتس = 9:16.
- **الصوت**: صمّمه يشتغل **صامت** (الكتابة كافية). أضف V.O عربي بلهجة فلسطينية/جليلية للدفء، أو موسيقى فقط. تجنّب موسيقى محقوق عليها — استعمل مكتبة منصّة الإعلان.
- **الـ CTA**: بدّل `contractorpro.app` بالرابط الحقيقي قبل النشر (في `promo-reel.html`، مشهد ٧).
- **الخط**: Noto Sans Arabic (نفس التطبيق) — مضمّن عبر Google Fonts في الصفحة.
- **الألوان**: مأخوذة حرفياً من `C`/`GRAD` في `src/constants/index.js` (Amber/Orange Dark).
- **مدّة مثالية للإعلان المموّل**: ابدأ بنسخة ١٥ث (أرخص CPM وأعلى إكمال)، واستعمل ٢٨ث لإعادة الاستهداف (retargeting).
- **القياس**: ضع نسخة لكل خطّاف (§٦) وأطفئ الأضعف بعد ٤٨ ساعة.
