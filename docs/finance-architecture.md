## 🎯 الرؤية النهائية

**ما هو هذا القسم:**
نظام إدارة مالية عملي لمقاول يدير أكثر من مصلحة —
يسجّل مدخولاته ومخروجاته، يأرشف فواتيره الواردة للمحاسب،
ويولّد قسائم رواتب لعماله.

**ما ليس هذا القسم:**
- ❌ لا يولّد فواتير ضريبية — عندك تطبيق منفصل لذلك
- ❌ ليس نظام محاسبة معقد

**قواعد الربط:**
- العملاء والمشاريع مشتركين — الربط بهم اختياري في أي سجل
- كل سجل مالي مرتبط بمصلحة محددة

## منطق الحساب حسب نوع المصلحة

> ⚠️ كل نوع عنده منطق مختلف — هذا يؤثر على الـ UI والحسابات والتقارير

### `עוסק פטור` — معفي
```
المدخول  → المبلغ الكلي فقط (بدون מע"מ)
المصروف  → المبلغ الكلي فقط (بدون מע"מ)
الصافي   → مدخولات − مخروجات
التنبيه  → سقف سنوي ₪120,000 (لو تجاوزه → تنبيه للترقية)
تقارير   → لا יש"ם ولا מע"מ
```

### `עוסק מורשה` — مرخص
```
المدخول  → المبلغ + מע"מ 18% مجموع (يُعرض منفصلاً)
المصروف  → المبلغ + מע"מ المدفوع (حسب الفئة):
           مواد / بضاعة / أدوات  → 100% قابل للخصم
           وقود / سيارات          → 67% قابل للخصم
           رواتب / تأمين          → 0% قابل للخصم
ميزان    → מע"מ محصّل − מע"מ مدفوع = مستحق لمصلحة الضرائب
تقارير   → תדריך מע"מ كل شهر أو شهرين
```

### `חברה בע"מ` — شركة محدودة
```
المدخول  → نفس עוסק מורשה (مع מע"מ)
المصروف  → نفس עוסק מורשה + رواتب المالكين كمصروف شركة
الأرباح  → ضريبة شركات 23% (بدل ضريبة دخل شخصية)
الرواتب  → المالك موظف في شركته = اقتطاعات مختلفة
تقارير   → דוח מע"מ + דוח שנתי للشركة
```

### التأثير على الكود
```javascript
// كل حساب يمر عبر business_type
function calcFinancials(entries, business_type) {
  if (business_type === 'osek_patur') {
    // بدون مع"م، بدون خصومات، سقف ₪120k
  }
  if (business_type === 'osek_moreh') {
    // מע"מ 18%، خصومات حسب فئة المصروف
    // ميزان מע"מ شهري/ثنائي
  }
  if (business_type === 'hevra') {
    // نفس osek_moreh + ضريبة شركات 23%
    // رواتب المالكين كمصروف
  }
}
```
