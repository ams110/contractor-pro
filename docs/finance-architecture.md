## المستوى الثاني: هيكل الشاشات

```
FinanceScreen (Tab الرئيسي)
│
├── 💰 المدخولات  (IncomeTab)
│   ├── قائمة المبالغ المحصّلة
│   ├── إضافة مدخول جديد
│   └── فلترة بالمشروع / التاريخ
│
├── 💸 المخروجات  (ExpensesTab) ← موجود، يُحسَّن
│   ├── قائمة المصاريف بالتصنيف
│   ├── إضافة مصروف + رفع إيصال
│   └── ملخص بالفئات (Recharts)
│
├── 🧾 الفواتير  (InvoicesTab)
│   ├── فواتير صادرة (أصدرتها أنت)
│   ├── فواتير واردة (استلمتها)
│   ├── إنشاء فاتورة + معاينة PDF
│   └── حالة الفاتورة (مدفوعة/معلقة/متأخرة)
│
├── 👷 قسائم الرواتب  (PayrollTab)
│   ├── قسيمة لكل عامل / شهر
│   ├── ملخص: راتب - اقتطاعات - سلف = صافي
│   ├── توليد PDF للقسيمة
│   └── إرسال للعامل (WhatsApp/PDF)
│
└── 📁 ملف المحاسب  (AccountantTab)
    ├── اختر الفترة (شهر / ربع / سنة)
    ├── حدّد ما تريد تضمينه
    ├── معاينة الملخص المالي
    └── تصدير حزمة ZIP أو PDF موحّد
```

## المستوى الثالث: خارطة الميزات

| القسم | الميزات |
|-------|---------|
| 💰 **المدخولات** | تسجيل دفعة، ربطها بمشروع، تحديد طريقة الدفع (كاش/تحويل/شيك)، رفع صورة إثبات |
| 💸 **المخروجات** | تصنيف المصروف، رفع إيصال، تضمين ضريبة، ربط بمشروع أو عام |
| 🧾 **الفواتير** | إنشاء فاتورة احترافية، إضافة بنود، حساب مع/بدون VAT، تصدير PDF، تتبع حالة السداد، أرشيف قابل للبحث |
| 👷 **قسائم الرواتب** | توليد قسيمة تلقائية من بيانات العامل، تضمين السلف والاقتطاعات، طباعة/إرسال PDF |
| 📁 **ملف المحاسب** | تجميع كل المستندات بفترة زمنية، ملخص P&L بسيط، تصدير دفعة واحدة للمحاسب |
