## 🎯 الرؤية الكاملة

نبني شيئاً **أبسط من iCount** لكن عملي بالكامل:
- قسم المحاسبة **مستقل** عن المشاريع — لا كل مشروع يدخل هون
- إدارة **أكثر من مصلحة/شركة** من نفس المكان، لكل واحدة اسمها وأرقامها
- الربط بالمشاريع **اختياري** — تقدر تربط فاتورة بمشروع لو حبيت، مش إجباري
- الهدف: مقاول يدير مصالحه المالية بسهولة ويصدّر ملف لمحاسبه

## المستوى الأول: قاعدة البيانات (Supabase)

### الجدول الجذر — المصالح / الشركات

```
┌──────────────────────────────────────────────┐
│                 businesses                   │  ← ❶ الأساس
├──────────────────────────────────────────────┤
│ id (uuid)                                    │
│ user_id → auth.users                         │
│ name          — اسم المصلحة/الشركة          │
│ business_type — osek_patur/osek_moreh/hevra  │
│ reg_number    — رقم ع.م / ח.פ / شركة        │
│ vat_number    — رقم الضريبة (لو مختلف)      │
│ phone                                        │
│ address                                      │
│ logo_url                                     │
│ bank_name, bank_account, bank_branch         │
│ is_active     — المصلحة الحالية المختارة     │
│ created_at                                   │
└──────────────────────────────────────────────┘
```

> كل ما يلي مرتبط بـ `business_id` — كل مصلحة عالمها المنفصل

## المستوى الثاني: هيكل الشاشات

```
FinanceScreen
│
├── 🏢 [Business Switcher] — شريط علوي لاختيار المصلحة الحالية
│   ├── اسم + رقم المصلحة النشطة (dropdown)
│   ├── + إضافة مصلحة جديدة
│   └── كل شيء تحت = بيانات المصلحة المختارة
│
├── 💰 المدخولات  (IncomeTab)
│   ├── إجمالي الشهر الحالي (بطاقة)
│   ├── قائمة الإيرادات
│   ├── فلترة: شهر / ربع / مصدر
│   └── + تسجيل مدخول جديد
│
├── 💸 المخروجات  (ExpensesTab)
│   ├── إجمالي + رسم بياني بالفئات
│   ├── قائمة المصاريف
│   ├── فلترة: شهر / فئة / مشروع
│   └── + تسجيل مصروف + رفع إيصال
│
├── 🧾 الفواتير  (InvoicesTab)
│   ├── صادرة / واردة (sub-tabs)
│   ├── إنشاء فاتورة احترافية
│   ├── رقم فاتورة تلقائي لكل مصلحة
│   ├── تتبع الحالة: مسودة/أُرسلت/مدفوعة/متأخرة
│   └── تصدير PDF / إرسال واتساب
│
├── 👷 قسائم الرواتب  (PayrollTab)
│   ├── اختر العامل + الفترة
│   ├── حساب تلقائي (أيام × يومية - سلف)
│   ├── معاينة + توليد PDF
│   └── إرسال للعامل
│
└── 📁 ملف المحاسب  (AccountantTab)
    ├── اختر الفترة
    ├── ملخص P&L: مدخولات - مخروجات = صافي
    ├── قائمة الفواتير والإيصالات للفترة
    └── تصدير PDF موحّد
```

## المستوى الثالث: Business Switcher (الفكرة المحورية)

```
┌─────────────────────────────────────────┐
│  🏢 شركة المقاولات ABC  ▼              │
│  ע.מ 123456789  •  نشطة               │
└─────────────────────────────────────────┘

عند الضغط:
┌─────────────────────────────────────────┐
│  ✓ شركة المقاولات ABC                  │
│    مقاولات الشمال                       │
│    مشاريع الديكور                       │
│  + إضافة مصلحة جديدة                   │
└─────────────────────────────────────────┘
```

**لكل مصلحة رقم فاتورة مستقل:**
- ABC: INV-ABC-001, INV-ABC-002...
- الشمال: INV-SHM-001...
