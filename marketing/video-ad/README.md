# إعلان الفيديو — Contractor Pro (نسخة عمودية 9:16، 15 ثانية)

مرجع موجز لإعلان المنتج القصير المُحسَّن (`cp_15s_ar_v22.mp4`)، مبني بالكود (HTML/CSS + رندر
عبر Chromium) ومربوط ببحث السوق في [`docs/MARKET_RESEARCH.md`](../../docs/MARKET_RESEARCH.md).

## الملفات
| الملف | الوصف |
|------|-------|
| `ad.html` | مشهد الفيديو كامل (7 مشاهد) مُتحكَّم بها زمنياً عبر `window.seek(t)` — بلا CSS animation، كل شيء deterministic لرندر نظيف |
| `render.mjs` | يفتح `ad.html` في Chromium، يلتقط 450 إطار (30fps×15s)، ثم `ffmpeg` → MP4 + الموسيقى |
| `assets/` | شاشات التطبيق (pulse/workdays/payroll)، خط Cairo (OFL)، الموسيقى `music15.m4a` |
| `cp_15s_ar_v22.mp4` | الإخراج النهائي — 1080×1920، 15s، h264 + AAC |

## إعادة التوليد
```bash
# يتطلّب: node، ffmpeg، playwright chromium
NODE_PATH=$(npm root -g) FFMPEG=$(which ffmpeg) node render.mjs
```

## بنية الإعلان (مربوطة بنقاط الألم)
| الوقت | المشهد | ربطه بالبحث (§4 نقاط الألم / §7 صياغة الإعلان) |
|------|--------|---------------------------------------------|
| 0–3ث | **الخطّاف**: «الساعة 11:47 ولسّا قاعد تحسب بإيدك؟» + فقاعات (راتب/إيصال/سلفة/מע"מ) | وجع التشتّت اليدوي آخر الليل — يمسك الانتباه فوراً |
| 3–5ث | عدّاد **1,800₪** «المحاسب بياخد منك… وما بيدير ولا عامل» | المرساة الأقوى: «أرخص من محاسب» (§3) |
| 5–6.5ث | انتقال «خلص. في طريقة أسهل.» | جسر للحل |
| 6.5–9ث | **نبض المصلحة 87/100** «كل شغلك بشاشة وحدة — بالعربي» | التموضع الفريد: واجهة عربية (§5) |
| 9–11ث | **أيام العمل** «سجّل يوم العامل — والراتب بينحسب لحاله» | تعقيد رواتب العمّال (§4.2) |
| 11–13ث | **الرواتب والسلف** «محسوبة لحالها» | الخصومات/السلف التلقائية |
| 13–15ث | **CTA**: «جرّب مجاناً 14 يوم · سنة 990₪ · اللينك بالبروفايل» | التسعير الموصى + إزالة حاجز البطاقة (§3) |

## ما تحسّن مقابل النسخة القديمة (v21)
| المشكلة في v21 | الإصلاح في v22 |
|----------------|----------------|
| أول ~10 ثوانٍ **صامتة** (الموسيقى تطلع بالنهاية فقط) | موسيقى **متّسقة من الثانية صفر** (~−18dB) عبر `dynaudnorm` + loop |
| الشاشات **صور ثابتة** محبوسة | حركة Ken-Burns على كل شاشة + **عدّاد 1,800₪ تصاعدي** |
| **نصّان مكرّران** بنفس المعنى في كل مشهد | **كابشن واحد** نظيف لكل مشهد |
| أيقونة **إيموجي 👷** تبدو placeholder | **أيقونة SVG (خوذة Lucide)** على تدرّج العلامة |
| لقطة انتقال **فاضية** | توهّج برتقالي ناعم + نصّ واضح |
| **لا توجيه للنقر** على تيك توك | إضافة صريحة «↑ اللينك بالبروفايل» |

## الناقص (الجزء البشري) — للنسخة القادمة
هذا الفيديو **موشن-جرافيكس بالكامل** (بلا وجه/صوت بشري — طبيعة البناء بالكود). لرفع التحويل
لدى الجمهور العربي الذي يشتري بالثقة (§5)، يُنصح بنسخة فيها **3 ثوانٍ سيلفي حقيقي** + تعليق
صوتي بالعامية. سكربت التعليق المقترح:

> **(0–3ث، وجه بموقع شغل):** «الساعة 12 الليل… ولسّا بحسب رواتب العمال بإيدي؟ خلص.»
> **(3–5ث):** «المحاسب بياخد 1,800 بالسنة وما بيدير ولا عامل.»
> **(5–13ث، على الشاشات):** «هون كل شي بالعربي: يوم عمل بلمسة، الراتب والـמע"מ بينحسبوا لحالهم، والعامل بيشوف حسابه ويطلب سلفته بنفسه.»
> **(13–15ث):** «سنة كاملة بـ 990 — أرخص من نص سنة محاسب. جرّبه مجاناً، اللينك بالبروفايل.»

### اختبار A/B مقترح
نسخة **موشن-جرافيكس** (هذه) مقابل نسخة **سيلفي بشري** (تُصوَّر يدوياً أو عبر منصّة أفاتار AI)
— تُنشر على تيك توك بالعربي (القناة الأقوى والمنافسون غائبون عنها، §6) ويُقاس أيّهما أعلى تحويلاً.
