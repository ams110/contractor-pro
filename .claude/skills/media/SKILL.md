# سكيل الوسائط — صور وفيديوهات

ملفات المستخدم المرفوعة توصل عادة على مسار مثل:
`/root/.claude/uploads/<uuid>/<filename>`

## الصور (jpg / png / webp / gif)

- اقرأها مباشرة بأداة **Read** — بتنعرض بصرياً وبتقدر تحلّلها فوراً.
- إذا كانت صورة تصميم مرجعي: حدّد العناصر (ألوان/خطوط/حركات/تكوين) واربطها بهوية المشروع (`C`/`GRAD` في `src/constants/index.js` وkit الفخامة `src/ui/Premium.jsx`) قبل أي تنفيذ.

## الفيديوهات (mp4 / mov / webm)

أداة Read ما بتقرأ فيديو — استخرج كادرات بـ ffmpeg واقرأها كصور:

1. **تأكد من ffmpeg** (مش مثبّت افتراضياً بالساندبوكس):
   ```bash
   which ffmpeg || sudo apt-get install -y ffmpeg
   ```
2. **استخرج كادرات** بمعدل 1fps مصغّرة (480px) لمجلد مؤقت:
   ```bash
   mkdir -p /tmp/vidframes
   ffmpeg -y -v error -i "<مسار الفيديو>" -vf "fps=1,scale=480:-1" /tmp/vidframes/f%03d.jpg
   ls /tmp/vidframes
   ```
   - فيديو طويل (>2 دقيقة)؟ خفّف المعدل: `fps=1/3` (كادر كل 3 ثوانٍ).
   - محتاج تفاصيل دقيقة بلقطة معيّنة؟ أعد الاستخراج حولها بدقة أعلى:
     `ffmpeg -ss <ثانية> -i <ملف> -frames:v 1 -vf scale=900:-1 /tmp/frame.jpg`
3. **اقرأ كادرات مختارة** بـ Read (البداية/الوسط/النهاية أولاً، ثم وسّع حسب الحاجة) — لا تقرأ كل الكادرات دفعة وحدة.
4. **نظّف** بعد ما تخلص: `rm -rf /tmp/vidframes`.

## لقطات شاشة للتحقق البصري (إخراج)

- التحقق من الواجهة يتم بـ **Playwright MCP** (انظر CLAUDE.md §17): المتصفح يُثبّت بـ `npx playwright install chrome` عند الحاجة.
- اللقطات بتنحفظ بمجلد العمل — **احذفها قبل أي commit** (لا تدخل الريبو).
- المشروع يحتاج env للتشغيل المحلي: أنشئ `.env.local` بقيم stub لـ `VITE_SUPABASE_URL`/`VITE_SUPABASE_ANON_KEY` (الملف git-ignored) واحذفه بعد التحقق.

## إرسال ملفات للمستخدم

ملف ناتج (صورة معدّلة/تقرير/لقطة) بدّو يوصل للمستخدم؟ استعمل أداة **SendUserFile** — ذكر الملف بالنص ما بيكفي.
