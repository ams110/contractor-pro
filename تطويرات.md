# خطة التطويرات - Contractor Pro

---

## 1. نظام الأدوار والصلاحيات
**الأولوية: عالية**

### الفكرة
المالك يدعو أشخاصاً بإيميلهم ويحدد لكل شخص صلاحيات مخصصة.

### الصلاحيات القابلة للتفعيل/التعطيل لكل مستخدم:
| الصلاحية | وصف |
|----------|-----|
| مشاهدة المشاريع | يقدر يشوف قائمة المشاريع وتفاصيلها |
| إضافة/تعديل المشاريع | يقدر يضيف مشاريع جديدة أو يعدّل الموجودة |
| مشاهدة العمال | يقدر يشوف قائمة العمال |
| إضافة/تعديل العمال | يقدر يضيف عمالاً أو يعدّل بياناتهم |
| مشاهدة المصاريف | يقدر يشوف المصاريف |
| إضافة المصاريف | يقدر يضيف مصاريف جديدة |
| مشاهدة الرواتب | يقدر يشوف الرواتب والدفعات |
| إضافة الرواتب | يقدر يسجّل دفعات الرواتب |
| مشاهدة التقارير | يقدر يشوف التقارير والإحصائيات |
| حذف أي شيء | يقدر يحذف سجلات |
| إدارة المستخدمين | يقدر يدعو ويعدّل صلاحيات الآخرين |

### آلية العمل:
- انت كمالك تدعو شخص بإيميله من إعدادات التطبيق
- تختار له الصلاحيات بمفاتيح تشغيل/إيقاف
- هو يستلم إيميل دعوة ويسجّل دخول
- تقدر في أي وقت تعدّل صلاحياته أو تحذفه

## 2. نظام الإشعارات
**الأولوية: متوسطة**

- تنبيه عند اقتراب موعد صرف راتب عامل
- تنبيه عند وجود مصاريف غير مدفوعة
- تنبيه عند إضافة مستخدم جديد لبياناتك

---

## 3. تصدير التقارير
**الأولوية: متوسطة**

- تصدير تقرير المشروع كـ PDF
- تصدير كشف رواتب العمال كـ Excel
- تصدير ملخص المصاريف

---

## 4. النسخ الاحتياطي والاستيراد
**الأولوية: منخفضة**

- تصدير كل البيانات كملف JSON
- استيراد بيانات من النسخة القديمة (localStorage)

---

## 5. دعم أكثر من شركة/حساب
**الأولوية: منخفضة**

- المالك يقدر يدير أكثر من شركة أو مشروع منفصل تحت نفس الحساب
- تبديل سريع بين الشركات
